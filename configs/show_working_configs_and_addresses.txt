! Show/verification command pack for 16th.pkt
! Use this after applying full_16th_completion.txt so you can demonstrate:
! 1) ACL policy is active
! 2) Routing is working
! 3) Interface IP addressing is correct
! 4) Device running configurations are present

! -------------------------------------------------------------------
! On EACH ROUTER (repeat per router in topology)
! -------------------------------------------------------------------
enable
terminal length 0
show running-config
show ip interface brief
show ip interface
show ip route
show access-lists
show run | section access-list
show ip protocols
show cdp neighbors detail

! -------------------------------------------------------------------
! On EACH SWITCH (repeat per switch in topology)
! -------------------------------------------------------------------
enable
terminal length 0
show running-config
show vlan brief
show interfaces status
show mac address-table
show cdp neighbors detail

! -------------------------------------------------------------------
! On EACH PC / SERVER (Desktop > Command Prompt)
! -------------------------------------------------------------------
ipconfig /all
arp -a
ping <default_gateway_ip>
ping <required_remote_ip>

! -------------------------------------------------------------------
! What to present as "all working configs and addresses"
! -------------------------------------------------------------------
! - Router/Switch: show running-config
! - Router addressing: show ip interface brief
! - Routing status: show ip route
! - ACL status + hit counters: show access-lists
! - Host addressing: ipconfig /all
